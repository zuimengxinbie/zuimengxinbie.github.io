## 社区挖金一

### css

- 使用：not选择器

  除了最后一个元素外，所有元素都需要一些样式，使用 `not` 选择器非常容易做到。

  如下图所示：最后一个元素没有底边。

  ```css
  li:not(:last-child) {
    border-bottom: 1px solid #ebedf0;
  }
  ```

- 修改光标颜色

  ```css
  caret-color: #ffd476;
  ```

- 删除type ="number"  末尾的箭头

  ```css
  .no-arrow::-webkit-outer-spin-button,
  .no-arrow::-webkit-inner-spin-button {
    -webkit-appearance: none;
  }
  ```

- 节日置灰

  ```
  body{
  	filter: grayscale(1);
  }
  ```
