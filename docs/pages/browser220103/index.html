    【chrome】- 高级调试技巧

    chenqi 4/1/2022 browser

    chrome 浏览器作为前端童鞋的老婆，一起来看看我们的老婆有多厉害....

    # 前言

    chrome 浏览器作为前端童鞋的老婆，相信你一定不陌生。调页面、写BUG、画样式、看php片少了它整个世界都不香了。

    # 一、一键重新发起请求

    • 选中Network

    • 点击Fetch/XHR

    • 选择要重新发送的请求

    • 右键选择Replay XHR

    # 二、控制台快速发起请求

    • 选中Network

    • 点击Fetch/XHR

    • 选择Copy as fetch

    • 控制台粘贴代码

    • 修改参数，回车搞定

    # 三、获取选中的元素

    调试网页时通过Elements面板选中元素后，如果想通过JS知道它的一些属性，如宽、高、位置等怎么办呢？

    • 通过Elements选择要调试的元素

    • 控制台直接用$0访问

    # 四、截屏

    将超出一屏的内容如何截取

    • ctrl + p 调出浏览器搜索
